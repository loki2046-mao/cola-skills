# Loki's Design DNA

> 这不是设计规范，是你的视觉指纹。
> 规范告诉你"不能做什么"，DNA 告诉你"你天然被什么吸引"。
> 任何人/AI 读完这份文档，应该能做出"一看就是 Loki 的东西"。

---

## 1. 核心张力 — 你的视觉母题

所有 Loki 的作品里，都藏着一组对立：

**精密 × 体温**

赛博机甲是精密的，但小熊猫是温暖的。
赤铜色是工业的，但奶杏渐变是生活化的。
3D 渲染是数字化的，但生活真实感是模拟的。

这不是偶然，这是你的审美本能。你在所有设计选择中，都在找这个平衡点。

> 如果一个东西只有科技感没有温度，那不是 Loki。
> 如果一个东西只有温馨感没有锋利，那也不是 Loki。

---

## 2. 质感偏好 — 你被什么触感吸引

### 2.1 喜欢的质感

| 质感 | 描述 | 在你的作品里怎么用 |
|------|------|-------------------|
| **磨砂金属** | 不是镜面，是带细微颗粒的哑光金属面 | 封面背景、卡片底色 |
| **暖光透射** | 光从物体背后或侧面透过来，边缘发亮 | IP 机甲发光、文字光效 |
| **纸张肌理** | 不是全新的纸，是翻过几遍、有轻微使用痕迹的纸 | CineType 背景、正文底色 |
| **玻璃折射** | 半透明、有厚度感、能隐约看到后面的东西 | 卡片叠加、信息层级 |
| **热气/蒸汽** | 刚煮好的咖啡、火锅的热气、冬天的呼吸 | 生活场景的氛围感 |

### 2.2 讨厌的质感

| 质感 | 为什么讨厌 |
|------|-----------|
| 塑料光泽 | 廉价感，像电商详情页 |
| 纯色平涂 | 没有呼吸感，像 PPT 模板 |
| 过度玻璃拟态 | 2022 年审美，已经过时 |
| 霓虹灯管 | 赛博朋克的陈词滥调 |
| 渐变彩虹 | 幼稚，没有节制 |

## 3. 光线语言 — 你偏爱的光

光是 Loki 作品里最重要的情绪工具。

### 3.1 首选光线

| 光线类型 | 情绪 | 使用场景 |
|---------|------|---------|
| **黄金时段侧光** | 温暖、怀旧、有故事感 | CineType 海报、封面 |
| **逆光轮廓** | 神秘、有距离感、电影感 | 观点文封面、行业观察 |
| **屏幕冷光 + 环境暖光** | 科技与生活的交界 | 工具测评、教程 |
| **单点聚光** | 专注、戏剧性、有分量 | 深度分析、思考文 |
| **漫射柔光** | 放松、日常、无压力 | 资源分享、生活类 |

### 3.2 光线禁忌

- 不要顶光（像审讯室）
- 不要均匀打光（像证件照）
- 不要彩色霓虹光（像夜店）
- 不要过曝（刺眼，没有层次）

## 4. 构图本能 — 你的空间感觉

### 4.1 你天然倾向的构图

| 构图方式 | 特征 | 你的使用习惯 |
|---------|------|-------------|
| **偏心构图** | 主体不在正中间，留出呼吸空间 | 封面、海报 |
| **前景遮挡** | 有东西在镜头前面虚化，制造纵深 | CineType、生活场景 |
| **文字压画面** | 文字不是"贴"在图上，而是"嵌"进场景 | 海报、封面 |
. **对角线引导** | 视觉元素沿对角线分布，有动感 | PPT、信息图 |
| **分层叠加** | 多层半透明元素叠在一起 | 卡片设计、UI |

### 4.2 你的空间禁区

- 上下左右完全对称（除非刻意做纪念碑感）
- 元素铺满整个画面（要有留白）
- 文字和背景没有对比度（看不清不如不放）

---

## 5. 色彩直觉 — 超越色值的东西

你选颜色不是在选色号，是在选温度。

### 5.1 你的色彩性格

| 属性 | 你的偏好 | 具体表现 |
|------|---------|---------|
| **色温** | 偏暖，但不闷 | 赤铜是暖的，但奶杏给了透气感 |
| **饱和度** | 中等偏低，不刺眼 | 很少用高饱和色做大面积 |
| **明度对比** | 喜欢深底亮字，或浅底深字 | 极端对比，不做灰色地带 |
| **色彩数量** | 一个项目最多 3 种颜色 | 主色+辅色+一个点缀，多了就乱 |
| **渐变方向** | 偏好从暗到亮，或从暖到更暖 | 封面渐变都是这个逻辑 |

### 5.2 你的色彩雷区

- 紫色系（除非是薰衣草紫做极小面积点缀）
- 荧光色（刺眼，没有质感）
- 灰蓝灰绿（没有性格）
- 红配绿（即使高级灰处理也不行）

## 6. 字体品味 — 你被什么字形吸引

### 6.1 你的字体审美

| 你被吸引的 | 你排斥的 |
|-----------|---------|
| 笔画饱满的黑体（有力量感） | 纤细的字体（没有存在感） |
| 有历史感的衬线体（有底蕴） | 花哨的手写体（不严肃） |
| 等宽字体（有技术感） | 圆体（太可爱，不专业） |
| 中英文混排时视觉重量匹配 | 中英文粗细差距太大 |

### 6.2 你的字体情绪映射

| 场景 | 字体感觉 |
|------|---------|
| 观点输出 | 厚重、有压迫感、不容忽视 |
| 教程干货 | 清晰、有条理、可信赖 |
| 生活分享 | 轻松、有手写感、不刻意 |
| 数据展示 | 等宽、精确、冷峻 |

## 7. 动态节奏 — 你的运动感觉

### 7.1 你喜欢的动态

| 动态类型 | 节奏 | 情绪 |
|---------|------|------|
| **渐入** | 慢，0.6-1s | 沉稳、有仪式感 |
| **交错出现** | 元素依次出现，间隔 0.1-0.2s | 有层次、不仓促 |
| **微弹** | 轻微 overshoot 后回弹 | 有活力但不夸张 |
| **视差滚动** | 前景和背景速度不同 | 有纵深、沉浸感 |

### 7.2 你讨厌的动态

- 弹跳过度（像卡通片）
- 旋转进入（没有意义）
- 所有元素同时出现（没有节奏）
- 闪烁/频闪（伤眼）
- 过渡时间超过 1.5s（太慢，浪费时间）

## 8. 材料语言 — 你作品里的"物"

### 8.1 你反复使用的视觉材料

这些元素在你的作品里反复出现，不是巧合，是你的视觉词汇：

| 材料 | 含义 | 出现频率 |
|------|------|---------|
| **光晕/光斑** | 希望、可能性、未来的暗示 | 极高 |
| **机甲/装甲** | 保护、专业、武装自己 | 高（IP 相关） |
| **纸张/书页** | 知识、沉淀、可触摸的信息 | 高 |
| **玻璃/透明** | 层次、复杂性、看得见但摸不着 | 中 |
| **热气/蒸汽** | 生活气息、正在进行时、鲜活 | 中 |
| **网格/坐标** | 结构化思维、技术底层 | 中 |
| **放大镜/镜头** | 探索、细节、观察者视角 | 低但精准 |

### 8.2 你从不使用的视觉材料

| 材料 | 原因 |
|------|------|
| 锁链/束缚 | 太沉重，不符合你的基调 |
| 火焰 | 太激进，你要的是温度不是爆发 |
| 翅膀 | 太鸡汤 |
| 齿轮 | 工业感过重，缺乏温度 |
| 地球/地图 | 太宏大叙事 |

## 9. 排版气质 — 你的文字空间感

### 9.1 你的排版直觉

- **段落短**：3 句以内，长了就切
- **留白大**：宁可少放内容，不要挤
- **层级狠**：标题和正文的大小差距要足够大，3:1 以上
- **左对齐为主**：标题可以居中，正文必须左对齐
- **列表简**：3-5 点，超过就拆

### 9.2 排版禁区

- 首行缩进（中文排版陋习，现代设计不用）
- 两端对齐（会出现不规则的字间距）
- 行间距低于 1.4（文字会粘在一起）
- 段间距小于行间距（分不清段落边界）

## 10. 情绪色谱 — 你的作品情绪地图

你的内容不是一种情绪，而是一个光谱。不同类型的内容对应不同的情绪位置：

```
冷 ←————————————————————————————→ 暖

行业观察    深度分析    工具测评    教程干货    资源分享    生活记录
  ↓           ↓           ↓           ↓           ↓           ↓
理性       思辨        客观        耐心        热情        放松
距离感     有锋芒      可信赖      手把手      想分享      无压力
```

每种情绪需要对应的：
- 色温（冷色 ↔ 暖色）
- 光线（硬光 ↔ 柔光）
- 字体重量（重 ↔ 轻）
- 信息密度（稀疏 ↔ 紧凑）
- 动态速度（慢 ↔ 快）

## 11. 一句话检验

做完一个设计后，问自己：

> "这个东西，换一个名字放在别人的公众号上，会有人觉得违和吗？"

如果会 → 这是 Loki 的。
如果不会 → 还不够独特，继续调。

## 2026-05-27 补充：Pinterest 参照图提炼

**核心修正**：Loki 的审美不是「只要高级克制」，而是「有态度、有质感、可拼凑」。
深底高级感和饱和活力感都可以用，关键是每个页面有明确构图意识，不能糊。

### 5条新增设计语言

1. **文字大字号层叠** — 字当图用。标题可以占满版面、和背景图融合叠压（参照 VAN GOGH INSIDE 海报）。文字不只是信息载体，是视觉主角
2. **构图优先** — 每个版面必须有明确构图形式：左右分 / 上下分 / 对角线 / 居中 / 散点。参照那张九宫格排版教学卡，构图意识本身就是设计语言
3. **中英文混排是信息分层工具** — 中文是内容，英文是标签/注释/分类。不是装饰，是帮读者快速定位层级的工具
4. **质感胜于扁平** — 哪怕不高级，也要有摄影感/材质感/层叠感。深蓝+咖啡杯叠压、暖色摄影+文字压图，都对。纯色扁平色块没有记忆点
5. **颜色敢用** — 深蓝、橙、暖米、高饱和都可以出现。但每个版面主色不超过2个，辅助色做点缀不做主角

### 从 Pinterest 提炼的可用组件手法

- **文字穿透背景图**（字母轮廓里有图像，VAN GOGH 风格）
- **大字序号装饰**（数字超大作为背景层，前景放内容）
- **摄影+文字叠压**（图片和标题之间没有安全区，直接叠）
- **棋盘/网格纹理背景**（增加质感，不用复杂图形）
- **产品图层叠溢出卡片边界**（不把图规规矩矩放在框里）
