## Loki 是谁（做设计之前必须先理解这个人）

**Loki，34岁，北京，中传研究生，ISTJ，现实主义执行者。**

她不追热点，只测真正有用的。她写过超多字的研究生论文，用同样的认真劲儿评测 AI 工具。她的公众号「赛博小熊猫Loki」5000+ 读者，每篇都是亲手跑过的，不水字数。

她喜欢苹果的克制高级，喜欢时尚杂志的排版张力，喜欢「穿Prada的女王」那种强悍气场。她对「普普通通」的东西有直觉上的排斥。

**她的第一原则：在 AI 时代，一个纯文科生是怎么用 AI 武装自己的。**
这是一个真实处境里的真实人在做真实的事——有具体身份（纯文科生）、有时代张力（AI 时代）、有主动性（武装自己）。
所有设计决策都从这里出发：不装技术专家，但比技术专家更懂得怎么用；不追热点，但比追热点的人走得更远。

**这决定了设计语言的一切。**

## 品牌 Persona（设计每一屏之前先问自己这三个问题）

### 1. 这个设计有没有「记忆点」？
Loki 的东西要让人看完说「哦，这是赛博小熊猫Loki的」。不是靠 logo，是靠气质。
- 有记忆点 = 某一处让人眼睛停下来，不是通篇平均用力
- 没记忆点 = 合格但普通，换一个品牌名也能用

### 2. 这个设计有没有「态度」？
Loki 说话直接，不废话，有自己的立场。
- 有态度 = 标题是「不追热点，只测真正有用的」
- 没态度 = 「关于我 · 个人介绍 · AI工具爱好者」

### 3. 这个设计有没有「克制感」？
克制不是简单，是「只保留必要的，必要的做到极致」。
- 克制 = 一屏只说一件事，说清楚，大量留白
- 不克制 = 把所有信息都塞进来，每个都想要

## IP 形象（设计语言的实体来源）

赛博小熊猫的 IP 形象：**赛博朗克 × 萌系卡通**，机械生物属性。

| 部位 | 颜色 | 设计系统可用 |
|------|------|------------|
| 头部 | 橙色 `#FF8C00`（高饱和正橙） | 大面积高亮度元素 |
| 耳朵 + 身体 | 深黑灰 `#1A1A1A` | 深底区块、重质感元素 |
| 脸部/腹部 | 象牙白 `#F5E6D3` | 文字底色、暖色调 |
| 胸前「LOKI」标志 | 青蓝霸光 `#00D9FF` | **点睛色**，少量使用但冲击力极强 |
| 胸口按鈕 | 红色 `#FF3333` | 极小量 CTA 或警示元素 |

**颜色即 IP，不可分割。** 每一次看到这套色，都在看那只小熊猫。

> **对设计局的启示：**
> - 练橙色 `#FF8C00` 滴入主色拼盘，与现有的琥珀橙 `#C87A45` 形成温度梯度（深底用琥珀橙，某些高光场景用练橙）
> - 青蓝霸光 `#00D9FF` 作为**点睛色**，出现一次就是记忆点，不能滥用
> - 深黑灰身体 = 深底区块，象牙白脸 = 文字区块，橙色头部 = 强调高亮

## 品牌气质关键词

**不是 AI 味 · 有态度 · 克制但有力 · 暖而不腻 · 技术底色 · 独立审美**

对立词（绝对不能出现）：
- ~~科技感~~ → 用「有温度的技术感」
- ~~商务专业~~ → 用「直接利落」
- ~~可爱卡通~~ → 用「质感 IP」
- ~~极简主义~~ → 用「克制而有力」

## 色彩系统

### 使用原则
**颜色是态度，不是装饰。** 每个颜色出现都有理由。

品牌橙是「点睛」，不是「主角」。一屏里橙色出现的地方不超过 3 处。
霸光青 `#00D9FF` 是 **P0 等级点睛色**，一屏最多出现一次，出就是记忆点。

### 品牌主色：琥珀橙

| 名称 | 色值 | 用途 |
|------|------|------|
| **琥珀橙（品牌色）** | `#C87A45` | 标题高亮词（1-2个词）、CTA、左竖线、标签边框、选中态 |
| 浅琥珀 | `#D9A87A` | 次级强调、hover 状态 |
| 深琥珀 | `#A05E30` | 深色场景强调、hover 深色版 |

### 底色系统

| 名称 | 色值 | 用途 |
|------|------|------|
| 暗底 | `#1A1816` | 深色背景（首屏 Hero、科技/产品类封面） |
| 纸白 | `#FDFAF5` | 文章正文区 |
| 暖米 | `#F5F0E8` | 页面大背景、教程类封面 |
| 燕麦 | `#EBE4D6` | 边框、分割线、标签底色 |

**深浅交替是必须的。** 连续两个深色区块 = 节奏死了。

### 辅助色（有明确语义，不可滥用）

| 名称 | 色值 | 语义 | 用途 |
|------|------|------|------|
| **霓虹青（点睛色）** | `#00D9FF` | IP胸前LOKI标志，冲击力最强 | 一屏最多一处：标志发光/高亮数字/点睛封面 |
| 练橙（IP头部色） | `#FF8C00` | 小熊猫头部，比琥珀橙更饱和 | 有IP形象场景配套，或活泼内容 |
| 烟青 | `#6B8590` | 冷色平衡，信息/客观 | 标签、hover、次要信息 |
| 苔绿 | `#7A8458` | 「My Take」观点色 | 个人评论标注 |
| 梅子 | `#7A5A6E` | 特殊/独特 | 极少量特殊场景 |

### 文字颜色

| 色值 | 用途 |
|------|------|
| `#FDFAF5` | 深底上的主文字 |
| `#1F1D1A` | 浅底上的主文字 |
| `#6B6157` | 副标题、描述 |
| `#9A9085` | 元信息、日期、muted |
| `#C4BBB0` | 极淡说明、装饰 |

### 禁用色（出现即返工）
- 蓝紫渐变（`#6C63FF`、`#a855f7`）
- Glassmorphism / 磨砂玻璃
- Neon 荧光色
- 纯黑 `#000000` 或纯白 `#FFFFFF` 大面积背景
- 科技蓝 + 银灰的「AI 工具通用配色」

## 字体系统 v3（得意黑方案）

### 四族字体

| 角色 | 字体 | 用途 |
|------|------|------|
| **标题** | 得意黑 `Smiley Sans Oblique` | 所有主标题、封面大字、金句 |
| **英文标签** | Space Grotesk | Eyebrow、标签行、英文装饰、数字大字报 |
| **正文** | Noto Sans SC | 正文段落 |
| **代码/期号** | JetBrains Mono | 代码块、日期标签、VOL 期号 |
| **竖排品牌** | LXGW WenKai | 偶尔用于竖排品牌名 |

### 字重对应规则（粗细反差是辨识度来源）

- 得意黑标题：不加 font-weight，用字体本身的斜体
- 副标题：`Noto Sans SC weight 100`——极细，反衬大标题
- 正文：`Noto Sans SC weight 300-400`
- Eyebrow 英文小标签：`Space Grotesk weight 300`，全大写，`letter-spacing: 0.18-0.22em`
- 数字大字报：`Space Grotesk weight 200`——越细越有张力

## 排版语言（Loki 的视觉说话方式）

### 必有的 3 个习惯

**① 中英交错 eyebrow**
每个内容区顶部都有一行 Space Grotesk 英文小标签：
```
CYBER PANDA · AI CREATOR · BEIJING
```
字号 10-12px，`letter-spacing: 0.18-0.24em`，颜色 `var(--smoke-light)` 或 `var(--muted)`。
这是节奏感的来源，不能省。

**② 标题关键词点橙**
标题中 1-2 个词用 `color: var(--amber)`，其余保持主文字色。
不是整行变色，是「某个词」——
```html
用 AI 做<span style="color:#C87A45">真正</span>有用的事
```

**③ 字号极端反差**
大标题极大（`clamp(48px, 7vw, 108px)`），下方说明极小极细（`14px weight 300`）。
中间没有「正常大小的标题」，那是最普通的选择。

### 留白是主动的选择
- 一屏只说一件事
- 主体周围的空白量 ≥ 内容本身的视觉重量
- 「感觉空了点」= 正确，「感觉刚好满」= 太满了

### 排版的「脊梁」
每一屏都有一条无形的轴线，内容沿这条线展开，不是随意摆放。
- Hero 区：底部左对齐，标题向上生长
- 数据区：网格线是脊梁，数字在格子里但不被格子困住
- 作品区：杂志式左右交替，脊梁是中轴线

## 组件选择逻辑（什么时候用什么）

| 场景 | 用什么组件 | 为什么 |
|------|-----------|--------|
| 「我是谁」 | 左侧数据列 + 右侧大字标题 | 克制地展示，不是自我介绍模板 |
| 数据/成就 | Space Grotesk 200 超细大字 + 极小说明 | 数字本身就是视觉元素 |
| 作品展示 | 真实截图做图片 + 杂志交替排版 | 真实感 > 漂亮的插图 |
| 金句/观点 | 左侧橙竖线 + 大字 + 极细署名 | 像杂志引用 |
| 工具标签 | 无底色边框 pill，烟青或橙色边 | 干净，不抢主体 |
| CTA 按钮 | 无圆角或极小圆角，琥珀橙底 | 有力量感，不软 |
| 分割线 | 1px 极细线，`var(--dark-line)` | 存在感低，但必须有 |

## 内容类型 → 视觉选择

| 内容类型 | 底色 | 封面风格 | 记忆点 |
|---------|------|---------|--------|
| AI 工具测评 | 暖米 | 浅底 A 款（右橙竖条） | 橙竖条是辨识点 |
| 工作流教程 | 暖米 | 浅底 A 款或 C 款（暗墨+网格） | 网格感 = 技术底色 |
| 观点/评论 | 全橙 `#C87A45` | 橙底白字 B 款 | 整屏橙 = 态度最强 |
| 作品展示/Landing | 深底 `#1A1816` + 浅底交替 | 杂志式排版 | 截图质感 + 极端字号 |
| 分享/演讲 Slides | 深底 + 浅底节奏交替 | 得意黑巨字 + 数据大字报 | 字号对比 |
