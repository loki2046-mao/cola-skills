# 质量检查清单 · checklist.md
> 做完任何设计，交付前逐条过一遍。P0 必须全过，P1 应过，P2 加分。

---

## P0 · 必须全过（任何一条没过就打回重做）

### 品牌一致性
- [ ] 品牌色只用了 `#C87A45`（及其深浅变体），没有出现其他彩色
- [ ] 没有蓝紫渐变、没有 Glassmorphism、没有 Neon 荧光色
- [ ] 背景是暖色系（`#F5F0E8` / `#FDFAF5` / `#1A1816`），不是纯白 `#FFF` 或纯黑 `#000`
- [ ] 文字是暖灰系，不是冷灰（如 `#888888`）

### 字体
- [ ] 主标题用了得意黑 `Smiley Sans Oblique`
- [ ] 正文用了 `Noto Sans SC 400`，没有用其他正文字体
- [ ] 英文标签/eyebrow 用了 `Space Grotesk` 或 `JetBrains Mono`
- [ ] 字体 CDN 链接都引入了（ZeoSeven + Google Fonts）

### 排版
- [ ] 没有用双引号 `""` → 改用【】
- [ ] 没有破折号 `——`
- [ ] 没有「不是A而是B」句式
- [ ] 没有四五个字一个句号的短句连击
- [ ] 没有 AI 味套话（赋能/抓手/闭环/接住了/稳/首先其次最后）

### 技术
- [ ] HTML 在浏览器里打开正常显示，字体加载成功
- [ ] 没有 console 报错

---

## P1 · 应该过（有理由跳过才行）

- [ ] 有至少一个视觉惊喜 section（不是每个区块都一样的节奏）
- [ ] 标题内有关键词换成 `#C87A45`（不是整行变色）
- [ ] 有粗细反差（得意黑大标题 + Noto Sans SC 100 极细副标题）
- [ ] 有中英交错节奏（Space Grotesk 英文小标签 + 中文内容）
- [ ] 背景装饰数字或引号（半透明，增加层次感）
- [ ] 封面有 Loki 头像 SVG

---

## P2 · 加分项

- [ ] 竖排品牌名（偶尔用，日杂感）
- [ ] 标签行字间距 `letter-spacing: 0.12-0.2em`，有呼吸感
- [ ] 深色面板打破整体节奏（如 Prompt 框的暗底）
- [ ] 截图框有 macOS 风格装饰头（三个小圆点）
- [ ] 浏览器截图后「一眼看出不是 AI 通用模板生成的」

---

## 封面专用检查

- [ ] 画布比例正确（公众号 900×383 / 小红书 3:4）
- [ ] 标题字号够大（公众号封面建议 ≥ 48px，小红书 ≥ 40px）
- [ ] 右侧品牌区（模板 A）或整底（模板 B）颜色是品牌橙 `#C87A45`
- [ ] 期号/标签用 JetBrains Mono

## 文章排版专用检查

- [ ] 预览宽度接近 560px（模拟微信阅读宽度）
- [ ] 底色是纸白 `#FDFAF5` 而非纯白
- [ ] Prompt 框有暗底代码区
- [ ] 结尾有 CTA 关注区

## 产品 UI 专用检查

- [ ] 侧栏底色 `#1A1816`，不是纯黑
- [ ] 当前选中态有左侧 2px 橙色竖线
- [ ] 按钮无大圆角（体现工具感）
- [ ] 品牌橙只用于激活/CTA，没有装饰性地随处散布
