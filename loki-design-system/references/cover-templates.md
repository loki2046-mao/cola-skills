# 封面模板规格 · cover-templates.md

---

## 公众号封面规格

- **画布**：900×383px（2.35:1）
- **字体加载**：见 brand-dna.md 字体加载代码
- **三种模板**：A（通用）/ B（观点/橙底）/ C（深色教程）

---

## 模板 A · 通用系列封面

**适用**：AI 工具测评、Skill 分享、系列更新、产品推荐（最通用，80% 场景用这个）

**结构**：左侧标题区（暖米底）+ 右侧品牌区（琥珀橙底，宽 200px）

```
┌──────────────────────────────┬────────┐
│ VOL.012 · AI实测              │  Logo  │
│                              │        │
│ [得意黑大标题，50px]          │ 赛博   │
│  副标题，18px #6B6157         │ 小熊猫 │
│                              │        │
│ [标签pill] [标签pill]         │ LOKI   │
└──────────────────────────────┴────────┘
```

**关键参数**：
- 左区底色：`#F5F0E8`
- 右区底色：`#C87A45`，宽 200px，flex-shrink 0
- 标题：得意黑 50px，最大宽度 520px
- 期号标签：`JetBrains Mono 14px #C87A45 letter-spacing 0.08em`
- 副标题：`Noto Sans SC 18px #6B6157`
- 标签 Pill：`JetBrains Mono 12px padding 4px 12px background #EBE4D6 color #3A342D`
- 背景装饰数字：得意黑 300px，颜色 `rgba(221,210,189,0.45)`，绝对定位右侧

---

## 模板 B · 观点/橙底封面

**适用**：个人观点、评论文章、金句类、有态度的内容

**结构**：全橙底，白色文字，顶部英文标签 + 中部大标题 + 底部作者

```
┌───────────────────────────────────────┐
│ OPINION — VOL.021          [Logo]     │  橙底 #C87A45
│                                       │
│     [英文小标签 Space Grotesk]         │
│     [得意黑大标题，52px]               │
│                                       │
│ [副标题，Noto Sans SC 100 14px]       │
└───────────────────────────────────────┘
```

**关键参数**：
- 整体底色：`#C87A45`，文字全白
- 英文标签：`Space Grotesk 300 12px opacity 0.5 letter-spacing 0.15em`
- 主标题：得意黑 52px
- 副标题：`Noto Sans SC 100 14px opacity 0.6`
- Logo：白色半透明（`rgba(255,255,255,0.7)`）

## 模板 C · 深色教程/暗夜封面

**适用**：步骤教程、深度分析、技术向内容

**结构**：暖墨底 + 网格纹理 + 左侧竖排品牌名 + 右侧标题

```
┌──┬─────────────────────────────────────┐
│竖 │ TUTORIAL — 5 STEPS                 │  暗墨底 #1A1816
│排 │                                    │  + 网格纹理
│品 │ [得意黑大标题，48px]                │
│牌 │  高亮词用 #C87A45                  │
│名 │                                    │
│   │ 副标题 #6B6157                     │
└──┴─────────────────────────────────────┘
```

**关键参数**：
- 底色：`#1A1816`
- 网格纹理（CSS）：`background-image: linear-gradient(rgba(255,255,255,0.015) 1px, transparent 1px), linear-gradient(90deg, rgba(255,255,255,0.015) 1px, transparent 1px); background-size: 24px 24px`
- 左竖排：`LXGW WenKai 13px #C87A45 writing-mode: vertical-rl letter-spacing 0.3em`，右侧有 1px 暗线分隔
- 英文 eyebrow：`Space Grotesk 10px #6B6157 letter-spacing 0.15em`
- 主标题：得意黑 48px，关键词换 `#C87A45`
- 副标题：`Noto Sans SC 100 13px #6B6157`

## 小红书封面规格

- **画布**：3:4 比例（如 1080×1440px 或等比缩小）
- **两种款型**：横评卡片 / 金句橙底

### 横评卡片

- 顶部：Logo + 名字 + "TESTED" 标签
- 标题：得意黑 40px
- 副标题：Noto Sans SC 100 14px
- 内容区：3 个排名卡，每卡含序号（JetBrains Mono 橙色）+ 名称 + 简评 + 星级
- 底部：「完整测评见正文 →」

### 金句橙底

- 整体底色：`#C87A45`
- 背景装饰：超大引号，`font-size 240px rgba(255,255,255,0.06)`
- 主文：得意黑 44px 白色
- 副文：`Noto Sans SC 100 18px opacity 0.65`
- 作者区：Logo + 名字 + 「关注 →」按钮（描边）
