# Prompt Templates

## Poster Template

主体描述 + 场景背景 + 构图方式 + 光影设置 + 材质细节 + 色彩策略 + 画质要求 + 禁止项。

## Toy / IP Template

角色设定 + 形体比例 + 表面工艺 + 配件系统 + 包装语义 + 拍摄式灯光 + 背景氛围 + 负面词。

## UI Template

产品类型 + 信息架构 + 网格与间距 + 组件层级 + 交互状态 + 动效节奏 + 品牌语气 + 可访问性与对比度约束。

## Negative 常用块

lowres, blurry, noisy texture, oversaturated, messy layout, extra fingers,
watermark, logo, typography artifacts, jpeg artifacts, cheap plastic look
