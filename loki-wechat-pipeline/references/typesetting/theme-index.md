# 主题索引与配色体系

本文件是配色的**单一来源**，与 loki-social / loki-deck 共用同一套配色体系（来源：`/Users/kude/.cola/skills/loki-social/references/themes.md`）。

## 设计理念

**配色与组件分离**——组件库只有一份（`components.md`），配色根据文章主题从下面12套里选，运行时替换变量。不再为每套配色写独立主题文件。什么样的主题适配什么配色，自己选。

## 辅助色变量（每套配色都有）

| 变量 | 用途 | 取值规则 |
|------|------|---------|
| `bg2` | 次级背景（卡片/引用块底） | bg 稍浅（深底）或稍深（浅底） |
| `text-sub` | 次级文字（图注/标签） | text 与 mute 之间 |
| `text-mute` | 辅助文字（时间/来源） | text-sub 再灰一档 |
| `border` | 细边框 | accent 的 10-12% 透明度 |
| `accent-soft` | 主色浅底（标签/提示块底色） | accent + 大量 bg 混合，或 accent 的 8-10% 透明度 |

## 配色选择决策树

```
文章题材？
├─ 科技/AI/工具测评 → 墨黑信号(01) 或 电光蓝黄(12) 或 灰蓝深度(05)
├─ 生活/心理/慢节奏 → 鼠尾草知识分子(08) 或 摩卡慕斯(07) 或 复古棕褐(09)
├─ 影评/文化/深度叙事 → 焦糖深蓝(04) 或 酒红夜幕(03) 或 黑色电影(06)
├─ 演讲/正式/高端 → 午夜金色(02) 或 墨黑信号(01) 或 黑色电影(06)
├─ 创意/年轻/梦幻 → 梦幻紫调(11) 或 摇滚墨水(10) 或 墨黑信号(01)
```

## 选择规则

- **用户已指定配色** → 直接用，不问
- **题材有明确契合** → 推荐：「这篇适合用 XX（理由），确认还是换一套？」
- **无明显倾向** → 默认推荐焦糖深蓝(04)（叙事兼容性最好）
- **全自动模式** → 自动选题材最契合的配色，交付时说明理由
- 同一篇文章只用一套配色，不混搭

## 深底与浅底的排版差异

**深底配色**（01-06, 11-12）：
- 公众号后台默认白底，深底需要用全局容器 `background: {{bg}}` 包裹
- 文字色用 `{{text}}`（浅色），注意深色模式兼容性天然好
- 主色 `{{accent}}` 在深底上更醒目，克制使用
- 图片包边用 `{{border}}`（半透明白色或主色）

**浅底配色**（07-10）：
- 公众号后台天然白底，浅底配色可以不包全局容器，或用极浅的 `{{bg}}`
- 文字色用 `{{text}}`（深色），注意可读性
- 主色 `{{accent}}` 做点缀，不要大面积铺
- 图片包边用 `{{border}}`（半透明深色或主色）

## 运行时变量替换

排版时，选定一套配色后，把 `components.md` 里所有 `{{bg}}`、`{{accent}}`、`{{text}}` 等变量替换为该套配色的实际色值。组件结构不变，只换颜色。
