# 视频位与关注引导图规则

## 一、视频位怎么做才稳

如果文章里有视频，不要期待“我写一个漂亮的外框，然后你去微信后台插入视频后，这个框还能完整包住播放器”。

原因很简单：

- 微信后台插入视频后，通常会生成平台自己的视频卡片
- 这个卡片不是普通图片节点，不会老老实实待在自定义框里
- 所以“包住视频本体的设计框”不稳定

最稳妥的做法是：

1. 先做一个视频说明卡
2. 说明卡里放封面图 / 截图 / 标题 / 一句导语
3. 在说明卡下面预留“这里去后台插入视频”

这样保住的是：

- 视频前的设计感
- 视频的上下文说明
- 整篇版式节奏

而不是硬保“视频播放器外框”。

## 二、文末关注引导图是硬需求

以后这套排版里，默认把“关注引导图”当成必选收尾模块。

执行规则：

1. 每篇文章最后必须有
2. 优先用原文档最后那张引导关注图
3. 如果原文档读不全，就用用户指定的固定尾图
4. 不允许因为重排而漏掉

当前默认固定尾图：

- `assets/signature.png`（相对于 SKILL.md 所在目录）

如果用户没有额外说明，默认直接使用这张图。

## 三、推荐工作流

如果你平时固定有一张“引导关注图”，最稳的方法是：

- 单独放一个固定文件
- 以后每篇文章排版时默认自动追加

这样就不会受飞书或 WPS 文档读取完整度影响。
