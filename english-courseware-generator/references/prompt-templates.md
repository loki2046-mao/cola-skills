## 提示词结构

每张课件图的提示词按以下结构组织：

```
1. 画面整体描述 + 风格声明
2. ZONE 布局描述（每个组件对应一个 ZONE）
3. 具体文字内容（逐字写入）
4. 色彩与风格约束
5. 反向提示词
6. 画幅比例
```

---

## 核心规则

### 规则1：所有文字逐字写死

提示词中出现的所有英文单词、中文翻译、题目、选项、对话内容，必须逐字写入提示词。

**正确**：
```
单词 "Rabbit" 用粗体字显示，下方写中文翻译 "兔子"。
```

**错误**：
```
显示词汇和翻译。  ← 模型会自由发挥，导致乱码
```

### 规则2：强调文字清晰可读

每条提示词必须包含：
```
4K分辨率，所有文字清晰可读，无乱码，无错字，字体清晰锐利
```

### 规则3：ZONE 分区不重叠

每个组件占据独立的画面区域，ZONE 之间有明确分隔（留白、分隔线或色块）。

### 规则4：风格全图统一

整张图使用统一的色彩系统、字体风格、边框样式。不能一个组件圆角一个组件直角。

### 规则5：文字量控制

每个 ZONE 内的文字量要适中。文字太多会导致生图模型文字模糊或乱码。如果文字量大，减少组件数量或增加画幅尺寸。

## 组件提示词片段

每个组件在 ZONE 2 中的描述片段。根据选用的组件组合拼接。

### 词汇卡片 Vocab Cards
```text
ZONE 2a — 词汇卡片区：
{{card_layout}} 排列的圆角卡片。每张卡片包含：
- 上半部分是 {{style}} 风格的可爱插画
- 英文单词 "{{WORD1}}" 用粗体字显示
- 下方用较小字体显示中文翻译 "{{翻译1}}"
- 统一的柔和色背景，圆角，柔和阴影
卡片内容：[{{WORD1}}/{{翻译1}}]、[{{WORD2}}/{{翻译2}}]、[{{WORD3}}/{{翻译3}}]、[{{WORD4}}/{{翻译4}}]
```

### 漫画对话 Comic Dialogue
```text
ZONE 2b — 漫画对话区：
横向 {{panel_count}} 格漫画，使用 {{theme}} 主题角色。
每格有简洁场景背景和圆角对话气泡。
第1格：{{Character A}} 说 "{{dialogue_1}}"
第2格：{{Character B}} 说 "{{dialogue_2}}"
第3格：{{Character A}} 说 "{{dialogue_3}}"
对话气泡为圆角，文字清晰可读。角色可爱有表情。
```

### 填空题 Fill in the Blanks
```text
ZONE 2c — 填空练习区：
顶部圆角框内是词库："{{word1}}  {{word2}}  {{word3}}  {{word4}}  {{word5}}  {{word6}}"
下方6道编号填空题，空格用下划线表示：
1. {{sentence_1_with_blank}}
2. {{sentence_2_with_blank}}
3. {{sentence_3_with_blank}}
4. {{sentence_4_with_blank}}
5. {{sentence_5_with_blank}}
6. {{sentence_6_with_blank}}
空格用清晰的下划线标示。题号用小圆圈数字。
```

### 选择题 Multiple Choice
```text
ZONE 2d — 选择题区：
{{question_count}} 道题，每题3个选项（A、B、C）：
Q1: {{question_1}}
   A. {{option_1a}}   B. {{option_1b}}   C. {{option_1c}}
Q2: {{question_2}}
   A. {{option_2a}}   B. {{option_2b}}   C. {{option_2c}}
[...继续所有题目]
选项用圆圈内字母标注。题与题之间间距清晰。
```

### 连线题 Matching
```text
ZONE 2e — 连线配对区：
左列：{{pair_count}} 个单词/图片
右列：{{pair_count}} 个翻译/图片（打乱顺序）
左侧内容："{{word1}}"、"{{word2}}"、"{{word3}}"、"{{word4}}"
右侧内容："{{trans1}}"、"{{trans2}}"、"{{trans3}}"、"{{trans4}}"
用虚线连接。项目间距清晰。
```

### 判断题 True or False
```text
ZONE 2f — 判断题区：
{{question_count}} 道判断题，带 ✓ 和 ✗ 图标：
1. {{statement_1}}   [ ✓ ]  [ ✗ ]
2. {{statement_2}}   [ ✓ ]  [ ✗ ]
[...继续所有题目]
图标为清晰的勾和叉。
```

### 图文配对 Picture-Word Match
```text
ZONE 2g — 图文配对区：
左侧：{{pair_count}} 个 {{theme_items}} 的可爱插画
右侧：{{pair_count}} 个英文单词框
用虚线连接图片和单词。
图片内容：{{picture_descriptions}}
单词："{{word1}}"、"{{word2}}"、"{{word3}}"、"{{word4}}"
```

### 单词搜索 Word Search
```text
ZONE 2h — 单词搜索区：
{{grid_size}} 字母网格，内藏单词。
网格中包含这些单词：{{word_list}}
字母网格（逐行）：

{{grid_row_2}}
[...所有行]
旁边小框内显示单词列表。网格有粗圆角边框，每格有细边框。
```


```text

3. {{example_3}}
4. {{example_4}}

```

### 语法点 Grammar Points
```text
ZONE 2k — 语法点区：
主题：{{grammar_topic}}
圆角框内简明讲解："{{explanation_text}}"
带颜色标注的例句：
1. {{example_1}}
2. {{example_2}}
3. {{example_3}}
语法关键部分用 {{highlight_color}} 高亮。
```

### 阅读理解 Reading Comprehension
```text
ZONE 2l — 阅读理解区：
文本框内有一段短文：
"{{passage_text}}"

下方题目：
1. {{question_1}}（选择题：A. {{a}} B. {{b}} C. {{c}}）
2. {{question_2}}（判断题：✓ / ✗）
3. {{question_3}}（简答：_______________）

短文有清晰的段落格式。题目编号。
```

## Negative Prompt

所有课件生成均附加以下反向提示词：

```text
无乱码文字，无错别字，无随机字母，无模糊文字，不可读的文字，文字区域不重叠，字体风格不统一，同一文字区域内不混用语言，无PPT模板感，无廉价练习卷感，无低分辨率文字，无水印，无logo，无不适当内容，无暴力元素，无恐怖元素，无扭曲插画，无杂乱排版，无文字溢出，无文字超出指定区域，无商业字体模仿
```
