# 排版填充法则 — 解决"下半部空白"问题

> 这是从用户提供的7张参考图中提炼的排版法则。核心问题：内容不多时，下半部大面积留白。核心解法：没有空白，只有没填满的区域。

---

## 六大填充法则

### 1. 三段式垂直分配（最重要）

每张卡片必须分成三段，每段都有内容：

```
┌─────────────────┐
│  顶部 25%        │ ← 标题/eyebrow/标签
│                 │
├─────────────────┤
│                 │
│  中部 50%        │ ← IP形象/主视觉/正文/数据
│                 │
├─────────────────┤
│  底部 25%        │ ← 辅助信息/装饰元素/总结框/页码
│                 │
└─────────────────┘
```

**禁止**：只填顶部，中部和底部空着
**必须**：三段都有视觉内容，哪怕底部只是装饰元素

### 2. IP形象作为视觉主体

IP形象不是小角标，是画面主角之一：

- **占据30-50%画面面积**
- **不是右下角小图**，而是中部/侧方的大图
- **与标题互动**：IP在标题旁边/下方，形成对话感
- **圆形/异形遮罩**：用圆形框、不规则裁切，不用方图

**封面标准布局**：
```
┌─────────────────┐
│  eyebrow        │
│  大标题         │
│  副标题         │
│                 │
│    [IP圆形大图]  │ ← 占画面30-40%
│                 │
│  meta数据       │
└─────────────────┘
```

### 3. 装饰元素填空

任何空白区域必须用装饰元素填充：

| 装饰元素 | 用法 |
|---------|------|
| 几何图形 | 圆点、三角、线条、方块散布在空白处 |
| 图案纹理 | 棋盘格、波点、条纹、网格底纹 |
| 重复文字 | 品牌名/主题词作为边框重复排列 |
| 色块分段 | 满版色块切割画面，不留纯色背景 |
| 小图标 | 相关icon散落在内容周围 |
| 虚线/实线 | 分隔线、引导线、装饰线 |
| 光晕/渐变 | radial-gradient填充空白区域 |

### 4. 色块满版分段

用满版色块切割画面，每个色块承载不同内容：

```
┌─────────────────┐
│ ████ 色块A ████ │ ← 标题区（深色块）
├─────────────────┤
│ ████ 色块B ████ │ ← 内容区（浅色块）
├─────────────────┤
│ ████ 色块C ████ │ ← 总结区（深色块）
└─────────────────┘
```

**优势**：色块本身就填满了画面，不存在"空白"

### 5. 图文叠压

文字和图片不分离，直接叠在一起：

- 文字叠在色块上
- 文字叠在图片上（加半透明遮罩保证可读性）
- IP形象部分被文字遮挡（增加层次感）
- 负空间切割：文字块推向边沿

### 6. Horror Vacui（恐空症）

**每个角落都有东西。** 但不杂乱，靠网格系统组织：

- 四角有装饰元素（小图标/几何形/标签）
- 边缘有重复文字边框或线条
- 底部有总结框/辅助数据/页码/品牌标识
- 中部空白处有光晕/纹理/装饰图形

## 各版式的填充策略

### 封面·居中大字
- 顶部：eyebrow标签
- 中部：大标题 + IP形象圆形大图（占30-40%）
- 底部：副标 + meta数据 + 品牌标识
- 装饰：光晕背景 + 噪点纹理 + 几何点缀

### 要点列表·编号卡片
- 顶部：eyebrow + 标题
- 中部：3-5个卡片用 `justify-content:space-evenly` 撑满
- 底部：总结框/装饰线/页码
- 每个卡片内部：标题+描述+可选小图标

### 金句·居中引用
- 顶部：大引号标记
- 中部：金句大字（占画面50%+）
- 底部：出处线 + 装饰元素
- 装饰：光晕 + 背景大字水印（得意黑装饰体）

### 杂志导读·双栏
- 顶部：eyebrow + 标题 + 分隔线
- 中部：双栏正文用 `align-content:center` 撑满
- 底部：总结框（border-left色条）
- 装饰：栏间竖线 + 段落间装饰符

### 卡片拼接·3横条
- 顶部：标题区（色块背景）
- 中部：3个横条用满版色块交替（bg/bg2/bg）
- 每条内部：序号+标题+描述 用 `justify-content:center` 居中
- 底部：最后一条自然到底

### 引用插入·长文
- 顶部：eyebrow + 标题
- 中部：正文+pull quote框 用 `justify-content:center` 撑满
- 底部：正文最后一段到底 + 品牌标识
- 装饰：pull quote框的色条边框 + 背景纹理

### 结尾·CTA
- 顶部：eyebrow
- 中部：CTA大字 + IP形象圆形大图
- 底部：副标 + 联系方式 + 品牌标识
- 装饰：光晕 + 噪点 + 装饰线
