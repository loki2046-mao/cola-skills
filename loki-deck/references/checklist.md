# 质量检查清单

## P0 · 必须通过（不过 = 返工）

- [ ] 浏览器 Tab 标题不是占位符
- [ ] 所有字体已加载（Noto Sans SC / Noto Serif SC / Space Grotesk / JetBrains Mono）
- [ ] 噪点纹理覆盖所有深底页面（`.noise` div 存在）
- [ ] 页码标识正确（01/10, 02/10...）
- [ ] 品牌标识（赛博小熊猫 SVG）在每页右下角
- [ ] 进度条在顶部正确递增
- [ ] 键盘翻页正常工作（← → / Space / ESC 索引）
- [ ] 没有文字溢出或重叠
- [ ] 深浅页面交替，没有连续 3 页同色
- [ ] 不用 emoji
- [ ] 颜色全部用 CSS 变量（不直接写 hex）

## P1 · 应该通过

- [ ] 标题用 Noto Sans SC 900（不是得意黑）
- [ ] 大数字用 Space Grotesk 200
- [ ] 眉头标签全大写 + 0.2em 字间距
- [ ] IP 形象在至少 2 个页面出现（封面+结尾推荐）
- [ ] 光晕效果在封面/数据/金句/结尾页存在
- [ ] 版式类型 ≥ 5 种（不要全是同一种）
- [ ] 章节过渡页至少 1 个

## P2 · 建议通过

- [ ] 关键词高亮每段不超过 3 个
- [ ] 图片比例标准（16:9 / 4:3 / 1:1）
- [ ] IP 形象状态与内容主题匹配
- [ ] 联系方式信息正确（公众号：赛博小熊猫Loki）
- [ ] 副标题用 Noto Serif SC 300 italic
