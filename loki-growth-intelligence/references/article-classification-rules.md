# 选题分类规则

## 选题类型分类标准

| 类型 | 判断标准 | 示例 |
|------|---------|------|
| 工具测评 | 以测某个 AI 工具/产品为主，有实测体验 | "我测了 XX 模型" |
| 教程实操 | 教读者怎么用某个工具/方法，步骤导向 | "用 AI 做 PPT 完整教程" |
| 行业观察 | 对 AI 行业趋势/事件的观察和分析 | "OpenAI 发布会后我想说" |
| 个人体验 | 作者自己的 AI 使用体验/故事 | "我用 AI 一个月后" |
| 选题迁移 | 把成功案例/方法迁移到新领域 | "用做公众号的方法做小红书" |
| 纯观点 | 表达观点/立场，无实操 | "AI 不会取代你" |

## 内容形式分类

| 形式 | 标准 |
|------|------|
| 长文 | > 2000 字 |
| 图文 | 文字 + 3 张以上图片 |
| 短文 | < 2000 字，图片 < 3 张 |

## 发布时段分类

| 时段 | 标准 |
|------|------|
| 深夜工作日 | 周一-周五 22:00-02:00 |
| 白天工作日 | 周一-周五 08:00-18:00 |
| 晚间工作日 | 周一-周五 18:00-22:00 |
| 周末 | 周六-周日 |

## 商单识别

| 标记 | 标准 |
|------|------|
| 商单 | 含品牌方合作内容，不论软硬 |
| 非商单 | 纯自发内容 |

## 统计规则

- 均值计算排除异常值（超过 3σ 的数据点）
- 样本量 < 10 的分类标注"统计意义有限"
- 增长率计算用 7 日移动平均，不用单日数据
- 转发率 = 转发数 / 阅读数
- 关注转化率 = 新关注数 / 阅读数
- 互动率 = (点赞 + 在看 + 评论) / 阅读数
