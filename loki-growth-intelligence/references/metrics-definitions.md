# 指标定义和计算公式

## 公众号核心指标

| 指标 | 公式 | 说明 |
|------|------|------|
| 阅读量 | 直接读取 | 文章发出后的累计阅读 |
| 转发率 | 转发数 / 阅读数 | 衡量传播力，Loki 历史均值参考 |
| 关注转化率 | 新关注数 / 阅读数 | 衡量涨粉效率 |
| 互动率 | (点赞 + 在看 + 评论) / 阅读数 | 衡量深度参与 |
| 商单浓度 | 商单数 / 总发文数 | 应控制在 30% 以下（Loki 要求每周 3-4 篇内容稀释商单） |

## 统计基准

- **均值**：同类型文章的算术平均
- **超预期**：高于均值 +1 标准差
- **符合预期**：均值 ±1 标准差范围内
- **低于预期**：低于均值 -1 标准差
- **扑街**：低于均值 -2 标准差

## 投产比指标

| 指标 | 公式 | 说明 |
|------|------|------|
| 单次使用成本 | 月费 / 月使用次数 | 每次调用花了多少钱 |
| 月投产比 | 关联产出价值 / 月费 | 产出 > 投入 = 正投产比 |
| 内容关联产出 | 该工具参与生产的内容数 | 如 ListenHub 参与了 3 期播客 |

## 增长指标

| 指标 | 公式 | 说明 |
|------|------|------|
| 周增长率 | (本周粉丝 - 上周粉丝) / 上周粉丝 | 衡量增长速度 |
| 获客成本 | 周投入 / 周新增粉丝 | 每个粉丝花多少钱 |
| 内容效率 | 周新增粉丝 / 周发布数 | 每篇内容带来多少粉丝 |
